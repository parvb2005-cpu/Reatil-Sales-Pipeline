# Reatil-Sales-Pipeline
This project is a Retail Sales ETL Pipeline built using Python, Pandas, NumPy, SQLite, and SQL. It reads sales, product, and store data from CSV files, cleans the data by removing missing and invalid values, and combines all the datasets into one final dataset. It also calculates the total revenue for each sale and stores the cleaned data in a SQLite database. Using SQL queries, the project generates useful business insights such as the top-selling products, total revenue for each store, and sales by city. To make the process easier, the entire workflow is automated using a run_pipeline() function, and basic error handling is included to prevent the program from crashing if any input file is missing. This project demonstrates practical skills in data cleaning, data transformation, SQL, database management, and ETL pipeline development.
